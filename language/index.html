Languages de programmation

Ecrire des algorithmes en français (sans aucun language) est n'importe quoi! Il vaut mieux apprendre un language de programmation pour apprendre à parler avec l'ordinateur, ensuite l'utiliser pour faire de l'algorithmique. L'idéal est de commencer par un language complet comme c++, c# ou java, pour balayer toutes les notions de programmation.
